## ✨ Features

- 🤖 **AI Post Generation** — Create engaging posts in seconds
- 🎨 **AI Image Generator** — Beautiful visuals powered by Google Gemini
- 📊 **Performance Tracking** — Mark posts as 🔥 hot, 😐 average, or 💔 flopped
- ✏️ **Smart Editor** — Edit and customize everything
- 🔐 **Secure Login** — Firebase authentication

---

## 🚀 Get Started

### Try It Now
👉 **[Open Post-Aura](https://post-aura.netlify.app)** — Sign up and create your first post!

### Local Setup (Optional)
```bash
git clone https://github.com/Ashurai84/Post-Aura.git
cd Post-Aura
npm install
cd frontend && npm run dev
```

Open `http://localhost:5173` in your browser.

---

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express
- **Database**: Firebase Firestore
- **AI**: Google Gemini API
