## 🚀 Ignition Sequence
```bash
# Clone the Engine
git clone https://github.com/Ashurai84/Post-Aura.git

# Install Dependencies
npm install
cd admin && npm install

# Fire the Thrusters
npm run dev:backend   # API Service
npm run dev:frontend  # User Dashboard
npm run dev:admin     # Command Center
```
